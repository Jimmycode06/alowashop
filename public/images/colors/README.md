# Pastilles de couleur (sélecteur de couleur produit)

Pour afficher une **image** dans la pastille au lieu du dégradé CSS, place les fichiers ici :

- **gold.png** – or
- **silver.png** – argent
- **onyx.png** – onyx (noir)
- **rose-gold.png** – or rose

**Format conseillé :** carré, par ex. 40×40 px ou 80×80 px (PNG ou JPG).  
Si un fichier est absent, la pastille utilise le dégradé CSS défini dans le code.
