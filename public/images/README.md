# Product Images

Place your ring product images here:

1. **ring-1.png** - Main product image (front view)
2. **ring-2.png** - Side view or different angle
3. **ring-3.png** - Detail view or close-up
4. **ring-4.jpg** - Lifestyle image (ring on hand, in use, etc.)

## Image Requirements:
- Format: JPG, PNG, or WebP
- Recommended size: 800x800px minimum (square format works best)
- File names: ring-1.png, ring-2.png, ring-3.png, ring-4.jpg

The images will automatically appear in the product gallery. If an image is missing, a placeholder will be shown.
